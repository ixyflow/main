# Premium TechNoir — Website

Static HTML/CSS/JS website for Premium TechNoir.

## Structure

```
premium-technoir-website/
├── index.html
├── inventory.html
├── styles.css
├── script.js
├── favicon.svg
├── CLAUDE.md
├── app.html
├── app.css
├── app.js
└── .gitignore
```

`index.html`, `inventory.html`, `styles.css`, and `script.js` make up the public Premium TechNoir website. The `app.*` files are a separate prototype and are not linked from the public site.

## Getting Started

Open `index.html` directly in a browser, or serve with any static file server:

```bash
npx serve .
```

## Initialize Git

```bash
git init
git add .
git commit -m "Initial commit"
```
